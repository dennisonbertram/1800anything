## GitHub Workflow

- **Labels**: bug, feature, enhancement, docs, agent, blocked
- **Project Board**: 1800anything (GitHub Projects v2)
- **PR Template**: `.github/PULL_REQUEST_TEMPLATE.md`
- **Issue Templates**: `.github/ISSUE_TEMPLATE/` (task, bug, feature)
- **Branch**: `main` is the default branch
- **PRs**: All changes should go through pull requests
